  • Each slot represent a one-hour match.

  • Busy hour's slots are the last 3 slots each day.
  • Each user has quota for 4 sub users that can be created by the user.
  • Each user has quota for only 2 playing times per week during busy hours, however can play during non-busy hours or be reserve for both busy hours & non-busy slots without limitation.
  • Total user & sub users has quota for only 4 playing times per week during busy hours, however can play during non-busy hours or be reserve for both busy hours & non-busy slots without limitation.
  • Each match could have 4 main players and 4 reserve players based on users quota as mentioned above.
  • Each user can only add/remove/switch slot for him/herself (No user or sub user should be able to allowed to modify for other users).
  • Match should close 4 hours before its playing time in case there are completed players which mean no changes are allowed, however match will keep open if there are less than 4 main players.
  • If a player reserved him/her self in the match as a main player, and any related sub user to this user can't add him/her self in the match as a reserve player but only as main player.
  • Green slot means that there is available place in the main players in this match
  • Yellow slot means that there is available in the reserve players and no available place in the main players in this match
  • Red slot means that there isn't available reservation in this match
  • Black slot means there isn't any match in this period
